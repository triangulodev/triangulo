Estudar CSS pode ser uma tarefa bastante complicada. São muitos
conceitos para aprender e muito conteúdo para consumir. O
**Guia de Estudos CSS** é um quadro no Trello que te
ajudará a organizar sua rotina de estudos e te dará o caminho das
pedras para ir do básico ao avançado no seu próprio tempo.

Este guia está dividido em alguns módulos, começando pelos
**Conceitos Básicos**, onde você aprenderá o
conhecimento base para utilizar CSS em seus projetos.

![Quadro no Trello do Guia de Estudos CSS](/images/guia-de-estudos-css/trello.png)

Em **Fundação sólida**, temos uma coletânea de tópicos
para solidificar sua habilidade em CSS. Após esta parte, você estará
apto para construir sites de quase todos os tipos e tamanhos.

Para complementar seu conhecimento, temos também
**Ferramentas** que te ajudarão a utilizar o CSS da
melhor forma possível, com o que há de mais moderno em
desenvolvimento frontend.

Em **Avançado**, cobriremos tópicos que te levarão
perto de dominar como utilizar o CSS em projetos frontend. Esta é
uma das seções que ainda crescerá muito em novas atualizações do
Guia.

Muitas pessoas utilizam frameworks JavaScript que fazem bom uso de
**CSS com JavaScript**. Neste módulo, apresentamos suas
ferramentas mais comuns.

Para finalizar, temos o módulo de **Frameworks**, onde
você aprenderá sobre alguns frameworks CSS que podem facilitar sua
vida em projetos frontend.

Este guia está em constante evolução. Se você tiver alguma
recomendação de conteúdo, envie no nosso
[Twitter](https://twitter.com/triangulo_dev) ou [Instagram](https://instagram.com/triangulo_dev).
Nos siga nestas redes sociais para receber novidades sobre o Guia
de Estudos CSS e outros conteúdos do Triângulo.
